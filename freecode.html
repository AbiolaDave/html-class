CSS Unit Guide: CSS em, rem, vh, vw, and more, Explained

Many CSS properties like width, margin, padding, and font-size take a length, and CSS has many different ways to express length.

.container {
font-size: 16px;
}
.container p {
font-size: 1em;
}
.container h2 {
font-size: 3em;
}
.container h3 {
font-size: 2em;
}
